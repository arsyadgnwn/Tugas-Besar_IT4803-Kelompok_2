# Tugas-Besar_IT4803-Kelompok_2
Rancanglah multi-linked list yang memodelkan film, aktor/aktris, dan hubungan di antara mereka. Seorang aktor dapat bermain peran di banyak film, dan satu film dapat diperankan oleh banyak aktor/aktris. Prgram ditujukan untuk mengelola daftar artis yang berperan dalam suatu film serta memantau performa nya
